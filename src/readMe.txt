# React Scientific Calculator

This is a scientific calculator built using React. It supports basic and advanced operations, handles multiple operations and parentheses, and includes a special feature where any operation involving 3 and 4 as operands triggers a confetti explosion using the `react-confetti-explosion` library.

## Features

- Basic arithmetic operations (+, -, *, /)
- Parentheses support
- Memory functions (MC, M+, M-, MR)
- Percentage (%)
- Negation (+/-)
- Scientific functions (sin, cos, tan, log, ln, sqrt, etc.)
- Degree and radian mode toggle
- Random number generation (Rand)
- Error handling
- Confetti animation for specific operations

## Instructions to Run the Project

Follow these steps to run the project on your local machine:

### Prerequisites

- Node.js installed on your machine
- npm or yarn package manager

### Installation

1. Clone the repository to your local machine:
    git clone <repository-url>

2. Navigate into the project directory:
    cd react-scientific-calculator

3. Install dependencies using npm
    npm install 

4. Run the app in development mode:
    npm start

5. Open http://localhost:3000 to view it in the browser.
